# 🍛 AI Recipe Generator

Pakistani AI Recipe Generator — Ingredients daalo, recipe milegi!

## 🚀 Deploy Karne Ka Tarika

### Step 1: GitHub pe Upload Karo
1. GitHub.com pe jao
2. "New Repository" banao — naam likho: `ai-recipe-generator`
3. Saari files upload karo (ya ZIP extract karke upload karo)

### Step 2: Vercel pe Deploy Karo
1. Vercel.com pe jao
2. "Add New Project" click karo
3. GitHub se apni repository select karo
4. **Framework: Vite** select karo
5. "Deploy" click karo — bas! ✅

### Step 3: API Key Lagao
Vercel Dashboard mein:
1. Settings → Environment Variables
2. Add karo: `VITE_ANTHROPIC_KEY` = your-api-key

## 💻 Local Run Karna
```
npm install
npm run dev
```
